MNIST DIGITS OCR

OPEN IN GITHUB

Project Overview:

   Using TensorFlow built a computer vision classfication model that predicts the subject of an input image. Using the MNIST Digits dataset built a model that identifies handwritten digits with high accuracy. The model uses convolutional layers that feature pooling before flattening and using an ANN to make predictions.

   Further expanding this project, currently have another version that features the Stanford OCR dataset. The model also makes use of a CNN that still requires hyperparameter tuning to improve.
